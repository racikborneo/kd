**Aplikasi Kamus Dayak Ahe / Kanayatn** adalah proyek *open-source* yang didedikasikan untuk melestarikan dan mendokumentasikan bahasa daerah melalui teknologi modern. Dibangun agar ringan, cepat, dan dapat diakses oleh siapa saja, di mana saja.

🚀 **[Lihat Demo Live](https://racikborneo.github.io/kd/)**

---

## ✨ Fitur Unggulan

- 📱 **Responsive Design**: Optimal untuk perangkat mobile, tablet, dan desktop.
- 🔍 **Fast Search**: Pencarian kata yang cepat berkat optimasi struktur data.
- 📂 **JSON Powered**: Data tersimpan rapi dalam format JSON, memudahkan pengolahan data.
- 🛠️ **Developer Friendly**: Kode bersih dan mudah dimodifikasi bagi pengembang lain.
- ☁️ **Zero Hosting Cost**: Berjalan sempurna di GitHub Pages.

## 🚀 Memulai Development (Installation)

Ingin menjalankan dan mengembangkan proyek ini di komputer lokal Anda?

1. Clone repositori ini
   ```html
   git clone https://github.com/racikborneo/kd.git
   ```   
2. **Masuk ke direktori proyek**
```html
cd kd
```
3. **Buka di browser**
Cukup buka file `index.html` pada browser favorit Anda.
